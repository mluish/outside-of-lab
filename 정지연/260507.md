### ✔️ runOnUiThread
> 백그라운드 스레드에서 UI(Main Thread) 작업을 실행할 때 쓰는 메서드

### 궁금한 점
- **Q1. UI 작업을 처리하는 다른 방법도 많은데 , runOnUiThread를 왜 쓰는지?**
    - A1. Activity가 UI Thread 접근용 편의 메서드를 제공한 것 (= Handler.post를 감싼 Activity 편의 메서드)
